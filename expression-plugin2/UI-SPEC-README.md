## 📖 文档内容

该文档详细量化了插件的所有 UI 参数，包括：

### 1. **结构说明**
- 完整的 UI 组件树状结构图
- 每个容器的层级关系

### 2. **参数配置**
所有组件的详细参数，包括：
- 窗口主容器参数
- 搜索面板参数
- 分类面板参数
- 表达式列表面板参数
- 编辑面板参数

### 3. **尺寸规范**
精确的尺寸数值：
- 窗口尺寸: `550×450`
- 左侧面板宽度: `160px` (最小 `140px`)
- 按钮宽度: `20-100px`（不同按钮不同）
- 列表高度: `120-200px`
- 编辑框高度: `80px`

### 4. **间距规范**
统一的间距标准：
- 窗口边距: `8px`
- 主区域间距: `6px`
- 面板边距: `[8, 12, 8, 8]`
- 组内间距: `4-6px`

### 5. **文本和图标**
完整的文本和图标映射表：
- 面板标题（📁 📝 ✏）
- 按钮文本（🔍 × ✏ ▶ 📋 💾 🗑）
- HelpTip 文案

### 6. **对齐方式**
常用对齐组合和应用场景

### 7. **代码示例**
每个主要组件的完整创建代码

### 8. **快速修改指南**
如何快速调整：
- 窗口尺寸
- 面板宽度/高度
- 间距
- 按钮宽度
- 文本和图标

## 🎯 使用方法

### 查看当前配置
打开 `UI-SPEC-REFERENCE.md`，查找你需要修改的组件部分。

### 修改 UI 参数

1. **找到对应组件**
   - 在文档中搜索组件名称（如 "搜索面板"）

2. **查看参数表格**
   - 每个组件都有详细的参数表格

3. **复制代码示例**
   - 文档中包含完整的代码示例

4. **修改参数值**
   - 按照表格中的说明修改数值

5. **测试效果**
   - 重启 AE 查看效果

### 快速修改常用参数

```javascript
// 窗口尺寸
win.minimumSize = [550, 450];  // 改为你需要的尺寸

// 左侧面板宽度
leftPanel.preferredSize.width = 160;  // 改为你需要的宽度

// 内容编辑框高度
expressionContentEt.preferredSize.height = 80;  // 改为你需要的高度

// 按钮宽度
button.preferredSize.width = 70;  // 改为你需要的宽度

// 间距
win.spacing = 6;              // 全局间距
mainGroup.spacing = 6;        // 面板间距
btnGroup.spacing = 6;         // 按钮间距
```
